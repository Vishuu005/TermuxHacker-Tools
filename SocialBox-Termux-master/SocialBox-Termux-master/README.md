# SocialBox-Termux
SocialBox is a Bruteforce Attack Framework [ Facebook , Gmail , Instagram ,Twitter ] , Coded By Belahsan Ouerghi Edit By samsesh
# Installation
```
apt-get update
apt-get install git
git clone https://github.com/Vishuu005/SocialBox-Termux.git 
cd SocialBox-Termux
chmod +x install-sb.sh
./install-sb.sh
```
# Run
```
./SocialBox.sh
```
